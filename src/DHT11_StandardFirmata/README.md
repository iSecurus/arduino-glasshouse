## Firmware

The provided Firmware DHT11_StandardFirmata is a modified version of the StandardFirmata included in the ArduinoIDE. The firmware was modified to handle the DHT11 sensor blocks created for Snap4Arduino. All modifications based on chrisclaynz project <a href="https://github.com/chrisclaynz/Robotics-unleashed">Robotics-unleashed</a>.
